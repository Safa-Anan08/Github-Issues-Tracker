### 1️⃣ What is the difference between var, let, and const?
### var works within the entire function, let works only within the block it is inside, and const works within that block but the value cannot be changed.
### 2️⃣ What is the spread operator (...)?
### The spread operator means to spread out. It allows to use all the items of an array or object separately.
### 3️⃣ What is the difference between map(), filter(), and forEach()?
### map() creates a new array to change values, filter() creates a new array to select items that meet a condition, and forEach() operates on each item but does not return a new array.
### 4️⃣ What is an arrow function?
### Arrow function is a short and simple way to write functions in JavaScript.
### 5️⃣ What are template literals?
### Template literals are strings written with backticks `, where variables or values ​​can be easily inserted with ${}.
